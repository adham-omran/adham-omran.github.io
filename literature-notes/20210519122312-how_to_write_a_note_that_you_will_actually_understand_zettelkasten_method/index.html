How to Write a Note That You Will Actually Understand • Zettelkasten Method

[Adham Omran] · May 10, 2021

source: How to Write a Note That You Will Actually Understand • Zettelkasten Method

Abstract

  • This is a summary of the original article, please read it before this one.

Write simple

  • Use paragraphs.
  • Use short understandable sentences.

Write with explicit form

  • Since notes can describe different items, from a philosophical idea to a statistics concept, different types of notes can exit with different form or template for each (YASnippet is a possible effective solution here.)

Use a clear title about the content

  • The title should describe the note.
  • Long titles are fine.

Add an abstract

  • Add an abstract that describes the content of the note.

Keep in mind

  • Notes should be written with a future you in mind, don’t write without making sure that you will understand in the future.
